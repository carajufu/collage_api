  학생 - 성적관리
	강의평가
  교수  - 성적관리
	강의평가
  관리자 - 성적관리
  
  학생 시험본 성적을 교수가 입력한다
  학생은 교수가 입력한 시험점수를 확인 한다
  
  과목별 성적에 각각 점수 만점은 100점
  100점에서 해당 성적 반영비율 (예:출석=25%)
  반영 성적비율로 총점을 구하고 해당 총점을
  https://www.konkuk.ac.kr/konkuk/2379/subview.do
  이 사이트의 반영 비율로 계산
  
  과목별 성적테이블의 한 학생이 여러 과목별성적의 점수와 학점이 나오면
  해당 학생의 점수 학점을 모두 불러와 학기별성적에
  https://www.konkuk.ac.kr/konkuk/2379/subview.do
  해당 학업성적평점평균계산예시의 계산식으로 해당 학기 총점과 학점평균이 나오면 된다
